# CLASbasicKoreanFlashcards
This is a simple web-app based from one of the many learning modules of CLAS Olongapo about learning the basics of Korean language. It shows simple flashcards to aid learners review an overview or some of the basic lessons digitally. 

##How to Download and Install the web-app on your phone (both works for iOS/Android)

- Simply copy the entire HTML source code of the index file (it contains internal CSS and Javascript, but don't mind, it will work).
- Open a new document or text file on your phone and paste the copied code and save as whatever you want to call the app and add '.html' as your file extension.
- Try to open the saved file on your phone and when prompted open it with whatever browser you have available on your phone. Done.
